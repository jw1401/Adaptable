Goals

The goal was to create a small online repository for adaptable 3D models that can be downloaded for 3D Printing. Users have the ability to upload 3D models in stl or jscad file format. Everyone who finds this site can adapt models and download them for personal use. Be aware of the model license which is based on creative commons (extarnal link).

Note

Please take note that this is a non commercial private website with the goal to develop a small site and learn programming Angular 4!
